# 05 - SQL Fundamentals

## Build
Learn relational database behavior by writing real queries against PostgreSQL or another relational database.

## What to implement
- Tables for a small domain
- CRUD queries
- JOINs and indexes
- Pagination and filtering
- EXPLAIN/ANALYZE experiments
- A few queries that intentionally get optimized

## Output
A database sandbox you can revisit later when performance questions come up.
