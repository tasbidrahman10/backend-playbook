# Python Data Model Lab

This lab is implementation-first. Build one exercise at a time and run it before moving on.

## Layout
- exercises/: focused scripts for each concept
- tests/: optional verification tests for behavior
- run_all.py: quick runner for all exercise modules

## Execution
Run one file at a time while building:
- python exercises/01_identity_vs_equality.py
- python exercises/02_repr_and_str.py

Run all tests when ready:
- pytest tests
