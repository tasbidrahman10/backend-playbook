# Phase 1 - Foundation

This folder is the working area for the first phase of the roadmap. The goal is to learn by building small systems that expose what frameworks, tools, and network layers are doing for you.

## Suggested order
1. Language core
2. How the internet actually works
3. Raw HTTP server
4. Linux and command line fluency
5. SQL fundamentals
6. Git beyond the basics

## Rule for this phase
Every topic should produce something concrete: a tiny tool, a lab, a script, or a server you can run and inspect.
