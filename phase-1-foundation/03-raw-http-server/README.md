# 03 - Raw HTTP Server

## Build
Implement an HTTP server without a web framework.

## Start small
- Accept a socket connection
- Parse the request line and headers
- Return plain HTML
- Return JSON
- Handle 404, 400, and 500 responses
- Add routing by path and method
- Add redirects and basic content types

## Why this matters
This is the clearest way to understand what frameworks hide from you.
